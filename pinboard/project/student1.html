Hector Buckberry: Object Poster

Unit 1: Starters

This project involved the analysis of an object and and creating a visual outcome.

This poster was created using Adobe Illustrator and features a 3D drawing of a plug. The poster explores the relationship between typography and space, using the plug as a central element to create a dynamic composition. The typography follows the pink line which represents the charging lead, creating a sense of movement and energy. The use of bold colors and geometric shapes adds to the overall impact of the design, making it visually striking and engaging.
